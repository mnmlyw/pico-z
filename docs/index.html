PICO-Z

Play any PICO-8 game on your desktop or in the browser. Save your progress anytime.

Play in Browser Download GitHub

Getting started

  1. Get a game — find .p8 or .p8.png carts on the PICO-8 BBS
  2. Open it — drag the cart file onto the PICO-Z window
  3. Save anytime — press P to save your progress, L to load it back

Controls

  • ↑↓←→ Move
  • ZC O button
  • XV X button
  • P Save state
  • L Load state
  • R Reload cart

Gamepads and mouse also supported.
